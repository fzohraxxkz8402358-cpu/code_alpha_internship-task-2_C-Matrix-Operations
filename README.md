# 🔢 C Matrix Operations

A command-line application developed in **C** that performs common matrix operations. This project demonstrates the use of **two-dimensional arrays**, **functions**, and **matrix algorithms**.

## 📌 Features

* ➕ Matrix Addition
* ✖️ Matrix Multiplication
* 🔄 Matrix Transpose
* User-friendly menu-driven interface
* Modular implementation using functions

## 🛠️ Technologies Used

* C Programming
* GCC Compiler (MinGW)
* Visual Studio Code

## 🚀 Getting Started

### Prerequisites

* GCC Compiler
* Visual Studio Code or any C IDE

### Compile

```bash
gcc matrix_operations.c -o matrix_operations
```

### Run

**Linux/macOS**

```bash
./matrix_operations
```

**Windows**

```bash
matrix_operations.exe
```
